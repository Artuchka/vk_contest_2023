# VK Contest 2023

## [🔗Live Demo🔗](https://vk-contest-2023.vercel.app/)

### 🚩Instructions🚩:

```bash
git clone https://github.com/Artuchka/vk_contest_2023.git

```

```bash
cd vk_contest_2023
```

```bash
npm install
```

```bash
npm run dev
```

to run Unit Tests:

```bash
npm run test:unit
```

to run E2E Tests:

```bash
npm run test:e2e
```
